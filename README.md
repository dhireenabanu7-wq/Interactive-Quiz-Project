# Interactive-Quiz-Project
Quiz Master Pro is a modern interactive quiz web application built using HTML, CSS, and JavaScript. It features multiple quiz categories, difficulty levels, timer-based questions, score tracking, achievements, leaderboard, dark mode, progress tracking, and result analytics using Chart.js.

# 🎯 Quiz Master Pro

A responsive and interactive Quiz Application built using HTML, CSS, and Vanilla JavaScript.

This project demonstrates front-end development concepts including DOM manipulation, event handling, timer management, localStorage, responsive UI design, and data visualization.

## ⚠️ Disclaimer

This is a frontend-only project created for learning and educational purposes.

- No backend integration
- No real user data collected
- Data stored locally using browser localStorage

## 🧠 How It Works

1. User enters their name.
2. Selects a category and difficulty level.
3. Quiz starts with a countdown timer.
4. Each correct answer increases the score.
5. Progress and performance are tracked throughout the quiz.
6. Final results display score, accuracy, achievements, and leaderboard rankings.

## 📊 Scoring System

- Correct Answer = +10 Points
- Wrong Answer = 0 Points
- Timeout = Treated as Incorrect
